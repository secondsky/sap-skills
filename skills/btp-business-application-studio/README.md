# BTP Business Application Studio Skill

Comprehensive guidance for SAP Business Application Studio (BAS), the cloud-based IDE on SAP BTP.

## When to Use This Skill

Use this skill when:
- Setting up SAP Business Application Studio subscriptions
- Creating and managing dev spaces
- Connecting to external systems (ABAP, S/4HANA, OData services)
- Configuring destinations with WebIDEEnabled properties
- Deploying MTA applications to Cloud Foundry
- Troubleshooting connectivity or dev space issues
- Managing roles and authorizations
- Working with Cloud Connector for on-premise access
